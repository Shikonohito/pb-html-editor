Заголовок первого раздела

Параграф, который содержит некоторое описание текущего раздела.

Разделы могут содержать несколько параграфов.

Разделов может быть несколько, но не могут быть вложенными.

Разделы идут друг за другом, имеют разделитель, который визуально отделяет их друг от друга.

Разделы могут содержать разные блоки. Блоки могут содержать заголовок и тело. Тело может быть универсальным или специализированным блоком. Тело может содержать такой же блок по структуре, в котором он находится.

⚙️ Заголовок блока с кодом только для копирования

Данный блок содержит контейнер с кодом, который можно скопировать, нажав кнопку в правой верхней части контейнера.

print("Hello, World!")

В данном блоке может быть 0 и более параграфов.

⚙️ Заголовок вложенного блока

Вложенность не должна превышать 2 уровня, так как усложнятся читаемость.

print("Hello, World!")

Параграфы могут быть между блоками.

⚙️ Заголовок блока с кодом для копирования и запуска

Данный блок содержит контейнер с кодом, который можно скопировать, нажав кнопку в правой верхней части контейнера. Также он содержит кнопку "Запустить" для выполнения кода. Результат выполнения отображается ниже, после кода.

print("Hello, World!")

                        ⚙️ Заголовок универсального блока
                        
                    

                    

                        
В заголовках этого блока могут быть эмодзи:
                            
    
                                
  • 📚 - означает определение термина.
    • 
                                
  • ⚙️ - означает объяснение алгоритма/кода/идеи/процесса.
    • 
                                
  • ⚠️ - означает предупреждение.
    • 
                                
  • 💡 - означает подсказка.
    • 
                                
  • ✅ - означает верный подход.
    • 
                                
  • ❌ - означает неверный подход.
    • 
                            

                        


                        
Тело может содержать 0 и более параграфов.


                        
Тело может содержать блоки с кодом только для чтения:

                        ❓ Заголовок блока с вопросом и единственным верным ответом
                        
                    

                    

                        

                            
Заголовок начинается с эмодзи ❓.

                            
Может содержать 1 и более параграфов. Для визуальной красоты в этом блоке может быть только 1 блок с вопросом.


                            
Может содержать вопрос с единственным верным вариантом ответа (радио-кнопки). При нажатии на вариант сразу же появляется фидбэк о том, почему ответ верный или неверный. Открытый фидбэк скрывается, если выбран другой вариант.

                            

                                
                                
                                
                            

                        

                    

                


                

                    

                        ❓ Заголовок блока с вопросом и несколькими верными ответами
                        
                    

                    

                        

                            
Заголовок начинается с эмодзи ❓.

                            
Может содержать 1 и более параграфов. Для визуальной красоты в этом блоке может быть только 1 блок с вопросом.


                            
Может содержать вопрос с несколькими верными вариантами ответ (чекбокс). При нажатии на вариант сразу же появляется фидбэк о том, почему ответ верный или неверный. Открытый фидбэк остаётся открытым, если выбран другой вариант, и скрывается только в том случае, если с варианта убирается галочка.

                            

                                
                                
                                
                                
                            

                        

                    

                


                

                    

                        ❓ Заголовок блока с открытым вопросом
                        
                    

                    

                        

                            
Заголовок начинается с эмодзи ❓.

                            
Может содержать 1 и более параграфов. Для визуальной красоты в этом блоке может быть только 1 блок с вопросом.


                            
Может содержать вопрос, поле для ввода ответа и кнопку "Ответить". При нажатии на кнопку появляется фидбэк. Если ответ неверный, то фидбэк покажет, что ответ неверный, и может содержать подсказку к верному ответу. Если же ответ верный, то фидбэк покажет, что ответ верный, и может содержать дополнительные объяснения к верному ответу. Проверяется точное совпадение, символ в символ.
