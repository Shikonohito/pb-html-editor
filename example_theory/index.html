Основы языка Python: от типов до функций

1. Введение

Python — это высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и читаемости кода. Синтаксис ядра Python минималистичен, что делает его отличным выбором для новичков.

Ключевая особенность Python — использование отступов (пробелов или табуляций) для выделения блоков кода, в отличие от фигурных скобок в C-подобных языках.

🐍 Код: Первая программа 
# Традиционная первая программа
print("Hello, World!")

2. Базовые типы данных

В Python все является объектом. Основные встроенные типы данных включают целые числа, числа с плавающей точкой, строки и логические значения.

🐍 Код: Примеры типов 
age = 25          # int (Целое число)
pi = 3.1415       # float (Число с плавающей точкой)
name = "Alice"    # str (Строка)
is_active = True  # bool (Логический тип)

3. Коллекции (Списки и Словари)

Python предлагает мощные встроенные структуры для хранения множества элементов.

Списки (List) — упорядоченные изменяемые коллекции. Элементы списка можно менять, добавлять и удалять.

🐍 Код: Списки 
fruits = ["apple", "banana", "cherry"]
fruits.append("orange") # Добавление элемента
print(fruits[0])        # Доступ по индексу (apple)

Словари (Dictionary) — коллекции пар «ключ: значение». Отличный выбор, когда нужно быстро находить данные по уникальному ключу.

🐍 Код: Словари 
student = {
    "name": "Bob",
    "age": 20,
    "courses": ["Math", "Physics"]
}
print(student["name"]) # Выведет: Bob

4. Условия и циклы

Программа редко выполняется строго линейно. Для ветвления логики используется if / elif / else. Для многократного повторения — циклы for и while.

🐍 Код: Условия и цикл for 
# Проверка условия
x = 10
if x > 5:
    print("x больше 5")
else:
    print("x меньше или равен 5")

# Цикл for по списку
for num in range(3):
    print("Итерация номер", num)
