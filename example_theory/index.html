01

Функции

Функция в Python — именованный блок кода, который выполняет одну логическую задачу. Она помогает убрать повторение, разделить программу на понятные части и описывать действия через имена.

Определение термина

Функция

Это переиспользуемый фрагмент программы с именем, входными данными и результатом. В Python функция создается через def, а вызывается по имени со скобками.

Важный момент

Одна функция - одна понятная задача

Если функция одновременно читает данные, считает результат и печатает отчет, ее сложнее проверять и менять. Лучше разделять такие действия на несколько функций.

Подсказка

Называйте функцию действием

Имя вроде calculate_total() или format_name() сразу объясняет, что делает код, поэтому вызов функции читается почти как предложение.

Верный подход

Возвращайте значение через return

Когда функция возвращает результат, его можно сохранить, проверить в тесте или передать в другую функцию без привязки к выводу на экран.

Неверный подход

Не прячьте результат только в print()

print() показывает текст пользователю, но не возвращает данные в программу. Из-за этого результат сложнее использовать повторно.

Аргументы передают функции данные. В Python можно использовать позиционные аргументы, именованные аргументы, значения по умолчанию, а также сбор произвольного количества значений через *args и **kwargs.

Область видимости

При поиске имени Python следует правилу LEGB: Local, Enclosing, Global, Built-in. Это объясняет, почему переменная внутри функции не всегда совпадает с переменной с тем же именем в модуле.

Lambda

Lambda-функция нужна для короткого выражения без отдельного имени. Ее удобно применять с sorted(), map() или filter(), когда логика помещается в одну строку.
