HTML Теория:
1. Адаптирована под Мобильный (портрет), Мобильный (ландшафт) / планшет, Планшет / небольшой десктоп, Десктоп, Широкий экран.
2. Разделена на левую (ширина 18rem) и правую части (всё доступное пространство).
3. Правая часть содержит верхнюю панель (высота 3rem, цвет фона #0a5a95), главный блок (цвет фона #f1f4f4) и может содержать блок самостоятельного изучения (цвет фона #f1f4f4) после главного блока.
4. Верхняя панель содержит оглавление урока (текст, высота 1rem, цвет текста #fbbf24), выравненный по левую сторону
5. Главный блок содержит тему урока (цвет текста #fbbf24) и разделы (максимальная ширина 920px).
6. Разделы содержат тему раздела (цвет текста #fbbf24) и наполнение. Между разделами имеется горизонтальный разделитель.
7. Наполнение может содержать обычный текст, картинки, анимации, видео, блок особый.
8. Блок особый имеет слегка скруглённую рамку (толщина 0.35rem, цвет рамки #0a5a95, цвет фона #0a5a95), содержит блок заголовка, внутри которого блок смайлика, затем правее блок текста (цвет текста #fbbf24), затем правее выровненная по правой части маленькая кнопка показать/скрыть (иконка сменяется от состояния открыт/скрыт). Ниже блока заголовка блок контекста (цвет фона #f1f4f4), тоже имеет слегка скруглённую рамку. Варианты блока особый:
8.1. Блок "вопрос-ответ": Смайлик вопроса, текст заголовка "Вопрос-ответ", изначальное состояние "открыт", блок контекста содержит блоки тестов, где требуется указать один или несколько ответов, или в виде поле ввода, куда нужно вводить ответ. Если в виде теста, то при выборе ответа плавно возникает обратный отзыв по выбранному варианту с объяснением, почему этот вариант верный (текст зелёный), или же почему этот вариант неверный (текст красный). При выборе другого варианта обратный отзыв по ранее выбранным не пропадает. Если в виде поле ввода, то всегда одно и то же объяснение, но если верно, то зелёным цветом, если неверно, то красным.
8.2. Блок кода: смайлик питона, текст заголовка "Код", изначальное состояние "открыт", блок контекста содержит блок кода (фон #1e1f22, тонкая рамка светлого оттенка, только для чтения), при наведении курсора на этот блок кода в правой верхней части появляется маленькая кнопка копирования, при наведении на эту кнопку она слегка темнеет и появляется надпись "Copy", при нажатии на неё копирует блок кода, в котором находится, визуально плавно меняет состояние кнопки после нажатия на статус "скопировано" и через 3 секунды состояние плавно меняется обратно.
8.3. Блок песочницы: подходящий смайлик под программирование, текст заголовка "Песочница", изначальное состояние "открыт", блок контекста содержит подключённую стороннюю API, с помощью которой можно программировать через веб-страницу.
9. Блок самостоятельного изучения содержит разделы (максимальная ширина 920px).
10. Разделы блока самостоятельного изучения содержат тему раздела и наполнение по такой же логике, как и разделы главного блока.
11. Левая часть в виде боковой панели (цвет фона #0a5a95), изначально видна на небольших десктопах, обычных десктопах, широких экранах, скрыта на планшетах, мобильный (ландшафт), полностью скрыта на мобильный (портрет), содержит верхнюю панель (высота 3rem), блок с лого (высота 150px), блок с подписью создателя сайта (текст высотой в 1.25rem, цвет текста #f1f4f4), разделитель с текстом "Содержание" (цвет текста #afafaf) и блок содержания (всё остальное).
12. Верхняя панель содержит кнопку 32x32px, выровненная по правую сторону. При нажатии на эту кнопку левая часть плавно уменьшается в левую сторону до ширины в 3.05rem (на мобильный (портрет) полностью скрывает левую панель, но кнопка видна). При повторном нажатии на эту кнопку левая часть плавно раскрывается обратно.
13. Блок содержания содержит блоки оглавлений, в котором находится название раздела (цвет текста #f1f4f4, выровненный по левую сторону), позволяет быстро переходить на раздел, нажимая по блоку оглавлений. Если выбран конкретный блок оглавления, то в самой правой части появляется треугольник влево (цвет #f1f4f4) выравненный по правую сторону. Если же левая часть в состоянии скрыто, блоки оглавлений не видны. Если же левая часть в состоянии полного сокрытия, то левой части просто нет.
