1. Всё должно находится в виде одного html файла practice.

2. Цель html файла - донести до студента практические задания в простом структурированном формате, который можно легко отправлять по сети, удобно читать, содержит интерактивные моменты и самопроверку кода.

3. CSS и JS должны подключаться из github.

4. При закрытии и новом открытии файла, изменения не сохраняются.

5. Адаптирована под Мобильный (портрет), Мобильный (ландшафт) / планшет, Планшет / небольшой десктоп, Десктоп, Широкий экран.

6. Разделена на левую (ширина 18rem) и правую части (всё доступное пространство).

7. Правая часть содержит верхнюю панель (высота 3rem, цвет фона #0a5a95) и главный блок (цвет фона #f1f4f4).

8. Верхняя панель содержит блоки иконок, выравненных по левую сторону.

9. Блок иконки может быть в трёх состояниях:

9.1. Неизвестно (цвет фона #5e5e5e, в центре прозрачный символ вопроса (svg)) - задание не выполнено.

9.2. Ошибка (цвет фона красный, в центре прозрачный символ крестика (svg)) - была нажата кнопка на проверку и автотесты дали ошибку.

9.3. Выполнено (цвет фона зелёный, в центре прозрачный символ галочки (svg)) - была нажата кнопка на проверку и все автотесты пройдены.

10. Главный блок содержит название задания (цвет текста #fbbf24) и разделы (максимальная ширина 920px).

11. Разделы содержат тему раздела (цвет текста #fbbf24) и наполнение. Между разделами имеется горизонтальный разделитель.

12. Наполнение может содержать обычный текст и блок особый.

13. Блок особый имеет слегка скруглённую рамку (толщина 0.35rem, цвет рамки #0a5a95, цвет фона #0a5a95), содержит блок заголовка, внутри которого блок смайлика, затем правее блок текста (цвет текста #fbbf24), затем правее выровненная по правой части маленькая кнопка показать/скрыть (иконка сменяется от состояния открыт/скрыт). Ниже блока заголовка блок контекста (цвет фона #f1f4f4), тоже имеет слегка скруглённую рамку. Варианты блока особый:

13.1. Блок "вопрос-ответ": Смайлик вопроса, текст заголовка "Вопрос-ответ", изначальное состояние "открыт", блок контекста содержит блоки тестов, где требуется указать один или несколько ответов, или в виде поле ввода, куда нужно вводить ответ. Если в виде теста, то при выборе ответа плавно возникает обратный отзыв по выбранному варианту с объяснением, почему этот вариант верный (текст зелёный), или же почему этот вариант неверный (текст красный). При выборе другого варианта обратный отзыв по ранее выбранным не пропадает. Если в виде поле ввода, то всегда одно и то же объяснение, но если верно, то зелёным цветом, если неверно, то красным.

13.2. Блок кода: смайлик питона, текст заголовка "Код", изначальное состояние "открыт", блок контекста содержит блок кода (фон #1e1f22, тонкая рамка светлого оттенка, только для чтения), при наведении курсора на этот блок кода в правой верхней части появляется маленькая кнопка копирования, при наведении на эту кнопку она слегка темнеет и появляется надпись "Copy", при нажатии на неё копирует блок кода, в котором находится, визуально плавно меняет состояние кнопки после нажатия на статус "скопировано" и через 3 секунды состояние плавно меняется обратно.

13.3. Блок самопроверки: подходящий смайлик под самопроверку, текст заголовка "Самопроверка", изначальное состояние "скрыт", блок контекста содержит подключённую стороннюю API, с помощью которой можно вписывать код, запускать через отдельную кнопку и проверять себя, прошёл ли все автотесты или нет.

14. Левая часть в виде боковой панели (цвет фона #0a5a95), изначально видна на небольших десктопах, обычных десктопах, широких экранах, скрыта (но можно раскрыть свапом или по кнопке) на планшетах, мобильный (ландшафт), на мобильный (портрет), содержит верхнюю панель (высота 3rem), блок с лого (высота 150px), блок с подписью создателя сайта (текст высотой в 1.25rem, цвет текста #f1f4f4), разделитель с текстом "Задания" (цвет текста #afafaf) и блок содержания (всё остальное).

15. Верхняя панель содержит кнопку 32x32px, выровненная по правую сторону. При нажатии на эту кнопку (или свап влево) левая часть плавно уменьшается в левую сторону до ширины в 3.05rem (на мобильный (портрет) полностью скрывает левую панель, но кнопка видна). При повторном нажатии на эту кнопку (или свап вправо) левая часть плавно раскрывается обратно.

16. Блок содержания содержит блоки оглавлений заданий, в котором находится блок иконки (логика и визуал как в правой части верхней панели) и правее название задания (цвет текста #f1f4f4, выровненный по левую сторону), позволяет быстро менять отображаемое задание, нажимая по блоку оглавлений. Если выбран конкретный блок оглавления, то у иконки появляется тонкая рамка (цвет #f1f4f4), а в самой правой части появляется треугольник влево (цвет #f1f4f4) выравненный по правую сторону. Если же левая часть в состоянии скрыто, то остаются только иконки (на мобильный (портрет) полностью скрывает левую панель).

Шрифт обычного текста: Inter (Sans-serif)
Размер обычного текста: 16–18 px
Межстрочный интервал: 1.5–1.7

Шрифт кода: JetBrains Mono (monospace)
Размер кода: 14–16 px
Межстрочный интервал: 1.5–1.7
