# Техническая спецификация: HTML-шаблон для теоретических уроков

## 1. Общие архитектурные принципы
- **Формат:** Один-единственный HTML-файл (включает в себя всю разметку).
- **Стили и Скрипты:** Подключаются извне. Для избежания проблем с MIME-типами и CORS используется хостинг на **Github Pages** (или CDN, обращающийся к Github).
- **Медиа-файлы:** Изображения, видео и анимации встраиваются в виде прямых внешних ссылок (требуют интернета), чтобы HTML-файл оставался лёгким и быстро скачивался.
- **Управление контентом:** Верстка контента происходит на чистом HTML внутри файла.
- **Сохранение состояния:** Строго **отсутствует**. При обновлении страницы (F5) или её переоткрытии весь прогресс (код в песочнице, ответы в тестах) сбрасывается.

## 2. Типографика и Цветовая схема
- **Шрифт текста:** Inter (Sans-serif), размер 16–18 px, межстрочный интервал 1.5–1.7.
- **Шрифт кода:** JetBrains Mono (monospace), размер 14–16 px, межстрочный интервал 1.5–1.7.
- **Основные цвета:**
  - Синий (основной фон панелей, рамок): `#0a5a95`
  - Светло-серый (фон контента): `#f1f4f4`
  - Желтый (заголовки, акценты): `#fbbf24`
  - Темный (фон кода): `#1e1f22`
  - Серый (разделители): `#afafaf`
  - Зеленый (верный ответ): `#22c55e` (или аналогичный читаемый)
  - Красный (неверный ответ): `#ef4444` (или аналогичный читаемый)

## 3. Макет и Адаптивность
Макет состоит из двух основных частей: Левой панели (сайдбар) и Правой части (контент).

### Брейкпоинты:
1. **Широкий экран / Десктоп / Небольшой десктоп:** 
   - Левая панель раскрыта по умолчанию (ширина `18rem`).
   - Кнопка скрытия находится в верхней панели левого меню. При клике панель сужается до `3.05rem`. Текст оглавления при этом скрывается.
2. **Планшет / Мобильный (ландшафт):** 
   - Левая панель скрыта по умолчанию.
   - Поведение аналогично десктопу, панель открывается поверх правого контента.
3. **Мобильный (портрет):**
   - Левая панель скрыта полностью (уезжает за экран).
   - Свайпы **отключены**.
   - Для открытия используется **плавающая кнопка (Fixed FAB)**, так как верхняя панель полностью скрывается.
   - При клике по ссылке оглавления на мобильном/планшете, меню **автоматически закрывается**.

## 4. Левая панель (Навигация)
- **Цвет фона:** `#0a5a95`
- **Структура сверху вниз:**
  - **Верхняя панель** (высота `3rem`): Содержит кнопку гамбургера `32x32px`, выровненную по правому краю.
  - **Логотип:** Блок высотой `150px`.
  - **Подпись создателя:** Текст `1.25rem`, цвет `#f1f4f4`.
  - **Разделитель:** Линия с текстом "Содержание", цвет текста `#afafaf`.
  - **Блок оглавления:** Занимает всё оставшееся место, скроллится при необходимости.
- **Элементы оглавления:** 
  - Названия разделов выровнены по левому краю (цвет `#f1f4f4`).
  - При клике плавно скроллит к соответствующему разделу.
  - Активный раздел помечается треугольником, направленным влево, выровненным по правому краю (цвет `#f1f4f4`).

## 5. Правая часть (Контент)
- Занимает всё доступное пространство справа от левой панели.
- **Верхняя панель:** Высота `3rem`, фон `#0a5a95`. Содержит статичный текст - название урока (`1rem`, цвет `#fbbf24`), выровненный по левому краю. На мобильных устройствах данная панель всегда остается видимой (sticky).
- **Главный блок:** Фон `#f1f4f4`. Содержит главный заголовок темы урока (`#fbbf24`).
- **Разделы:** Максимальная ширина `920px`, центрируются. Содержат заголовки раздела (`#fbbf24`) и сам контент (текст, медиа, особые блоки). Разделяются горизонтальной линией.
- **Блок самостоятельного изучения:** Располагается под главным блоком, фон `#f1f4f4`, структура разделов аналогична главному блоку.

## 6. Особые блоки контента
Все особые блоки имеют слегка скруглённую рамку `0.35rem` цвета `#0a5a95`, фон заголовка `#0a5a95`.
Шапка блока содержит: Смайлик → Название блока (`#fbbf24`) → Кнопка свернуть/развернуть (справа).
Контекстный блок (тело) имеет фон `#f1f4f4` и скруглённые углы. Блоки изначально открыты (развернуты).

### 6.1. Блок "Вопрос-ответ" (Тесты)
- **Заголовок:** Смайлик ❓, текст "Вопрос-ответ".
- **Логика ответов:** Хранятся открыто в HTML (например, атрибут `data-correct="true"`). Никакой криптографии.
- **Один вариант (Radio):** При клике на вариант сразу появляется обратный отзыв (зеленый или красный текст с объяснением). Предыдущие отзывы не пропадают при переключении.
- **Несколько вариантов (Checkbox):** Моментальный отзыв после каждого клика по чекбоксу. Кнопка "Проверить" не требуется.
- **Поле ввода:** Рядом с полем находится кнопка "Ответить". При клике на неё появляется отзыв (цвет зависит от правильности). Объяснение всегда одно и то же.

### 6.2. Блок "Код"
- **Заголовок:** Смайлик 🐍, текст "Код".
- **Контекст:** Фон `#1e1f22`, светлая тонкая рамка. Только для чтения.
- **Подсветка синтаксиса:** Используется собственный JS-скрипт на регулярных выражениях, который парсит код и подсвечивает базовые ключевые слова Python (без тяжелых библиотек вроде Prism).
- **Кнопка копирования:** Появляется в правом верхнем углу при наведении на блок. При наведении на саму кнопку появляется тултип "Copy", кнопка темнеет. При клике копирует код, текст меняется на "скопировано" на 3 секунды.

### 6.3. Блок "Песочница"
- **Заголовок:** Смайлик 💻, текст "Песочница".
- **Технологии:** Monaco Editor + Pyodide (исполнение Python прямо в браузере).
- **UX загрузки:** Так как WASM файлы Pyodide весят много, при первоначальной загрузке блока отображается красивый индикатор загрузки (спиннер). После инициализации загрузчик скрывается и появляется редактор.
